# BotAlphatJS
Bot Kicker And Bot Anti Kicker, Check Sider.

Cara Install Bot AlphatJS :
------
- `pkg install nodejs -y`
- `pkg install git -y`
- `git clone https://github.com/Nadyatjia/BotAlphatJS`
- `pkg install nano`
- `cd AlphatJS`
- `npm install`
- `npm start`

Cara Menjalankan Bot Kembali :
------
- `cd AlphatJS`
- `npm start`
